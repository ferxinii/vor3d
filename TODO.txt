[ ] refineregion (src/delaunay_constrained.c): current implementation always inserts the midpoint of edge (va,vb) and re-queues the face. TetGen's version (tetgen.cxx:18295) sets ivf.rejflag=1 so the insertion is rejected if M encroaches on any segment; in that case it splits the encroached segment instead (a different point, so DT topology always changes). Without this, if Level 1 and Level 2 both keep failing for the same face, Level 3 will insert the same midpoint repeatedly (scplx_insert_point_cdt returns -1 on duplicate → failure). Fix: before inserting M, check encroachment of M against all ridge edges using the diametric sphere test (reuse scout_refpt logic); if encroached, split that segment instead. Also: add the segment cascade (iterate subsegstack after insertion, as TetGen does at line 18338) for robustness on degenerate inputs. Both are only needed if Level 3 fires multiple times on the same face, which should not happen in practice once Level 2 (T3-T7) is in place.

[ ] ensure_ridge_protected (src/delaunay_constrained.c): current Steiner point placement uses the midpoint, which can diverge -- a vertex close to one endpoint keeps encroaching on the shorter sub-segment after each split, causing infinite halving toward that endpoint. The fix is TetGen's "scout + project" approach (tetgen.cxx, delaunizesegments ~L15893 and getsteinerptonsegment ~L15810):
    1. Walk DT tetrahedra from endpoint A toward B (scoutsegment ~L15612); along the path find the "reference point" refpt = vertex with the largest interior angle subtended by the segment (i.e. deepest inside its diametric circumsphere).
    2. Place the Steiner point at the orthogonal projection of refpt onto the segment, not the midpoint. Special case: if refpt lies on an adjacent segment sharing an endpoint, mirror refpt's distance from that endpoint. Clamp to [0.2, 0.8] of segment length as a fallback.
    3. After the split, refpt is no longer inside the circumsphere of either sub-segment, so both can be recovered without further splitting due to refpt. This guarantees termination.
